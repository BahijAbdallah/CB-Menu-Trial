Bug: Setting “Display Order” in admin doesn’t change item order in category pages.

  displayOrder (preferred) OR display_order.

- Do NOT alter components or styles. Only sort the array.

Insert this helper (shared utils or in the file):
  const orderVal = (it) => {
    const v = it.displayOrder ?? it.display_order ?? it.order ?? null;
    const n = typeof v === 'string' ? parseInt(v, 10) : v;
    return Number.isFinite(n) ? n : Number.POSITIVE_INFINITY;
  };
  const sortByDisplayOrder = (items) =>

      .map(x => x.it);
