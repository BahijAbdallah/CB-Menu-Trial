What to do

Find the allergens block (search for the heading text ALLERGENS in the component; it’s in menu.tsx or a small Allergens component).

Wrap it like this (use these class names/IDs so the CSS below applies):

<section id="allergens">
  <h2 className="title">ALLERGENS</h2>
  <p className="note">Please ask our staff for guidance on allergens and cross-contamination.</p>

  <div className="list">
    {ALLERGENS.map(a => (
      <div key={a.key} className="item">
        <img src={a.icon} alt={a.label} />
        <span className="label">{a.label}</span>
      </div>
    ))}
  </div>
</section>
