Update the hero section to match the reference mockup exactly.

What to do

Put the hero image at /public/images/hero.jpg. If the file is named differently (e.g. DSC00032-1.jpg), copy/rename it to /public/images/hero.jpg.

Move the logo into the hero (top-left) and make it bigger. Hide the header’s brand so we keep only the EN/Halal pills on the right.

Show the hero image as a full-bleed background (no patterns/overlays hiding it).

Swap fonts:

“The Menu” → Billmake

“OF BEYROUTH” → Parslay

Code changes

In the page/component that renders the top section (e.g. menu.tsx or Hero.tsx), replace the hero with:

<section className="hero" aria-label="Hero">
  {/* Top-left logo inside hero */}
  <img src="/images/logo.png" alt="Chez Beyrouth" className="hero-logo" />

  <div className="hero-inner container">
    <h1 className="hero-title">
      <span className="line-1">The Menu</span><br />
      <span className="line-2">OF BEYROUTH</span>
    </h1>
  </div>
</section>


In the header component/CSS, hide the brand inside the header (keep the EN/Halal pills):

/* hide brand only in header so we show the large logo inside hero */
.site-header .brand{ display:none !important; }


In public/styles.css (or global CSS), add/replace:

/* Ensure no site-wide pattern/overlay blocks the hero image */
body, .site-header, .hero{ background-image:none !important; }

/* HERO background */
.hero{
  position:relative;
  height:520px;
  background:url('/images/hero.jpg') right center / cover no-repeat !important;
}
.hero::before, .hero::after{ content:none !important; }

/* Top-left logo in hero */
.hero-logo{
  position:absolute; top:18px; left:22px;
  height:72px;               /* bigger logo */
}
@media (min-width:992px){ .hero-logo{ height:92px; } }

/* Title lockup bottom-left */
.hero-inner{ position:relative; height:100%; }
.hero-title{
  position:absolute; left:22px; bottom:28px;
  margin:0; color:#3b4b30; line-height:1.06;
}

/* SWAP FONTS: line-1 Billmake, line-2 Parslay */
.hero-title .line-1{
  font-family:'Billmake', sans-serif;
  font-size:38px; letter-spacing:.02em;
}
.hero-title .line-2{
  font-family:'Parslay', serif;
  font-size:38px; letter-spacing:.06em;
}

@media (max-width:768px){
  .hero{ height:400px; }
  .hero-title .line-1, .hero-title .line-2{ font-size:30px; }
}


If the image still doesn’t appear, hard-set the background inline (this overrides any stale CSS):

<section
  className="hero"
  aria-label="Hero"
  style={{
    backgroundImage: "url(/images/hero.jpg)",
    backgroundPosition: "right center",
    backgroundSize: "cover",
    backgroundRepeat: "no-repeat"
  }}
>
  <img src="/images/logo.png" alt="Chez Beyrouth" className="hero-logo" />
  <div className="hero-inner container">
    <h1 className="hero-title">

    </h1>
  </div>
</section>
