The menu categories (Breakfast Items, Salads, …) must appear in one horizontal row, centered, and scrollable sideways on both mobile and desktop. Do not change each chip’s existing colors/shape/typography.

Implement:

HTML (wrap existing chips):

<nav class="cat-scroll" aria-label="Menu categories">
  <ul class="cat-track">
    <!-- Move each existing category chip/link into an <li>. Keep inner classes/styles. -->
    <!-- <li>...Breakfast Items chip markup...</li> -->
    <!-- <li>...Salads chip markup...</li> -->
    <!-- repeat for all -->
  </ul>
</nav>


CSS (add; don’t delete chip styles):

/* viewport */
.cat-scroll{
  width:100%;
  margin:0 auto;
  overflow-x:auto;
  overflow-y:hidden;
  -webkit-overflow-scrolling:touch;
  scrollbar-width:none;       /* Firefox */
  scroll-snap-type:x proximity;
}
.cat-scroll::-webkit-scrollbar{ display:none; }   /* WebKit */

/* track: horizontal, centered */
.cat-track{
  display:flex;
  flex-wrap:nowrap;
  justify-content:center;     /* centered when not overflowing */
  align-items:center;
  gap:12px;
  padding:10px 16px;
}

/* items don’t shrink; snap nicely */
.cat-track > li{
  list-style:none;
  flex:0 0 auto;
  scroll-snap-align:start;
}

/* ensure chips behave inline (prevents vertical stacking) */
.cat-track > li > a,
.cat-track > li > button,
.cat-track > li > span,
.cat-track > li > div{
  display:inline-flex;
  align-items:center;
  justify-content:center;
}

/* optional: cap row width and keep it centered on large screens */
@media (min-width: 1024px){
  .cat-scroll{ max-width:1200px; }
}


(Optional) Desktop wheel → horizontal scroll:

<script>
document.addEventListener('DOMContentLoaded', () => {
  const row = document.querySelector('.cat-scroll');
  if (!row) return;
  row.addEventListener('wheel', e => {
    if (Math.abs(e.deltaY) > Math.abs(e.deltaX)) {
      row.scrollLeft += e.deltaY;
      e.preventDefault();
    }
  }, { passive:false });
});
</script>


Acceptance:

All category chips are beside each other in one row, centered.

The row scrolls horizontally via touch/drag; on desktop, wheel scroll moves sideways (optional JS).

No visual changes to individual chip styles.
